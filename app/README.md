# App Flutter

## Estado
Este folder esta listo para crear el proyecto Flutter cuando Flutter este instalado.

## Requisitos
- Instalar Flutter SDK y agregarlo al PATH
- Android Studio o Xcode (segun plataforma)

## Crear proyecto
1. Desde este folder:
   - flutter create zapato_ai_app
2. Entrar al proyecto:
   - cd zapato_ai_app

## Configuracion API
- Base URL: http://localhost:8000
- Endpoints:
  - POST /captures (multipart)
  - POST /recognize (multipart)
