# Roadmap

## Fase 1: Dataset y etiquetado

- Definir catalogo inicial y SKUs
- Captura de imagenes con guia
- Consolidar etiquetas y revisar calidad

## Fase 2: Baseline ML

- Modelo preentrenado + fine-tuning
- Evaluacion top-1/top-5 y confusion
- Exportar modelo a formato movil

## Fase 3: Backend

- API de catalogo/stock
- Endpoint de reconocimiento
- Busqueda de similares por embeddings

## Fase 4: App Flutter

- Camara y captura
- UI de resultados y stock
- Modo offline parcial

## Fase 5: Mejora continua

- Monitoreo de errores
- Retraining periodico
