# Esquema de etiquetas

## Campos

- sku: string (codigo unico)
- brand: string
- model_name: string
- type: string (formal, casual, deportivo, street)
- color_primary: string
- color_secondary: string (opcional)
- material: string (opcional)
- gender: string (opcional)

## Ejemplo JSON

{
"sku": "BAT-CA-0001",
"brand": "Bata",
"model_name": "Casual Air",
"type": "casual",
"color_primary": "negro",
"color_secondary": "blanco",
"material": "cuero",
"gender": "unisex"
}
